# Synopsis

This application flatten an array of arbitrarily nested arrays of integers into a flat array of integers.

e.g. [[1,2,[3]],4] -> [1,2,3,4].

## Dependencies:

`Ruby v2.2.2` or higher
`Ubuntu Operating System`

## RUN Application

```
make run
```
